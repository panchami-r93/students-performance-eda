# Students Performance – Exploratory Data Analysis

Exploratory data analysis (EDA) on a student performance dataset using Python.

## 🎯 Problem statement

Coaching centers and schools often don’t know which factors (study time, test preparation, parental education, etc.) actually impact students' exam scores.  
This project analyzes a students performance dataset to identify the factors most associated with high and low scores, so educators can focus on what really matters.

## 📂 Files

- `data/students_performance.csv` – dataset  
- `notebooks/students_performance_eda.ipynb` – main analysis notebook  

## 🧰 Tech stack

- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Jupyter Notebook / Google Colab  

## 🔍 What I did

- Loaded and explored the dataset  
- Checked data types, basic statistics, and missing values  
- Analyzed distributions of exam scores  
- Explored relationships between factors (e.g., test preparation, gender, race/ethnicity) and scores  
- Summarized key findings at the end of the notebook  

## 📌 Key findings (summary)

- Students who completed the test preparation course tend to have higher scores than those who did not  
- There are noticeable differences in scores between different race/ethnicity groups  
- Score distributions show a wide range between low and high performers

## 🧠 What this can help with

- Helping a coaching center decide where to invest time (extra classes, test prep, etc.)  
- Designing targeted support for students at risk of low scores
